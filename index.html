Quality of Experience (QoE)

Telecom operators used to gather QoE data from various technical srouces such as drive testing tools and network probes, which all fail to provide a truly complete picture of the end-user experience. In the age of big data, The innovative SIM-based QOE application tracks individual subscribers' mobility experience (calls, data service, etc.) while leaving their personal data private. By collecting the QOE data End-to-End, network operators can duplicate subscriber insights and carry out geographical network improvement.

Mastercard M/Chip Mobile (MCM)

Being the first NFC payment application certified by Master-Card, MCM application is also compliant with the other relevant specifications like GlobalPlatform Card, EMV, etc. It covers all card portfolios including debit and credit. Meanwhile, the UICC-embedded solution brings the highest level of chip security to the mobile payment market.

GlobalPlatform Secure Element Access Control (GPSEAC)

The GPSEAC application is designed to prevent unauthorized access to resources in the Secure Elements (UICC, microSD, etc). The security application allows critical banking and transport applications to be deployed on modern mobile devices, which often have insufficient security protection in their operating systems. This security mechanism is fully compliant with GPSEAC standard and can provide transparent control access to client applications running in the devices.

Fault-Tolerant Automotive E/E Architecture

Recent decades have witnessed a revolution in automotive Electrical and Electronic (E/E) architectures, where the complexity in design and analysis grows fast. Such systems are fault-free and their reliability management is crucial to the overall safety of vehicles. At the research platform TUM CREATE, we investigated the existing fault-tolerance strategies and created a deterministic and decentralized diagnosis method to address critical faults in automotive E/E architectures.

Real-Time Java for Industrial Automation

Organizations in industrial automation face the challenges in extending the lifetime of their embedded control applications. These applications, due to the increasingly short innovation cycles of hardware and platform software from 3rd party vendors, often have to be migrated from legacy systems to the new and more capable ones. To minimize the re-engineering cost in a long term, an advanced software migration concept is proposed based on the novel Real-Time Java technology.

A blogging application with RESTful API

A blog system is a typical web CRUD application that serves for object create, read, update and delete. Here, this small blogging application demonstrates how to build browsable RESTful APIs using Django REST framework for such a CRUD application. In addition, the frontend of the blog website is built with AngularJS to consume the RESTful APIs.
