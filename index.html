Quality of Experience (QOE)

Telecom operators used to gather QoE data from various technical sources such as drive testing tools and network probes, which all fail to provide a truly complete picture of the end-user experience. In the age of big data, The innovative SIM-based QOE application tracks individual subscribers' mobility experience (calls, data service, etc.) while leaving their personal data private. By collecting the QOE data End-to-End, network operators can duplicate subscriber insights and carry out geographical network improvement.

  • Company: Gemalto Pte Ltd, Singapore
  • Service: Mobile Subscriber Services

Mastercard M/Chip Mobile (MCM)

Being the first NFC payment application certified by MasterCard, MCM application is also compliant with the other relevant specifications like GlobalPlatform Card, EMV, etc. It covers all card portfolios including debit and credit. Meanwhile, the UICC-embedded solution brings the highest level of chip security to the mobile payment market.

  • Company: Gemalto Pte Ltd, Singapore
  • Service: Mobile Payment

GlobalPlatform Secure Element Access Control (GPSEAC)

The GPSEAC application is designed to prevent unauthorized access to resources in the Secure Elements (UICC, microSD, etc). The security application allows critical banking and transport applications to be deployed on modern mobile devices, which often have insufficient security protection in their operating systems. This security mechanism is fully compliant with GPSEAC standard and can provide transparent control access to client applications running in the devices.

  • Company: Gemalto Pte Ltd, Singapore
  • Service: Mobile Payment

Fault-Tolerant Automotive E/E Architecture

Recent decades have witnessed a revolution in automotive Electrical and Electronic (E/E) architectures, where the complexity in design and analysis grows fast. Such systems are fault-free and their reliability management is crucial to the overall safety of vehicles. At the research platform TUM CREATE, we investigated the existing fault-tolerance strategies and created a deterministic, decentralized diagnosis method to address critical faults in automotive E/E architectures.

  • Company: TUM CREATE, Singapore
  • Service: Automotive Electronics

Real-Time Java for Industrial Automation

Organizations in industrial automation face the challenges in extending the lifetime of their embedded control applications. These applications, due to the increasingly short innovation cycles of hardware and platform software from 3rd party vendors, often have to be migrated from legacy systems to the new and more capable ones. To minimize the re-engineering cost in a long term, an advanced software migration concept is proposed based on the novel Real-Time Java technology.

  • Company: ABB Corporate Research Center , Germany
  • Service: Embedded Software
